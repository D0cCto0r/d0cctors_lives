# D0cCtor's Lives

Mod privado para Minecraft/NeoForge 1.21.1.

Sistema de vidas diarias con:

- Fragmento del Ciclo
- HUD personal de vidas
- limbo
- reset diario
- comandos OP
- mensajes globales al morir
- efecto al respawnear tras perder una vida

## Compilar

```powershell
gradle clean build
```

El `.jar` compilado queda en:

```txt
build/libs/
```

## Versión base

```txt
D0cCtor's Lives 1.0.0-fix13
Minecraft 1.21.1
NeoForge 21.1.121
```
